h1. Starling Signals Extension - CommandSignal

The SignalCommandMap is an extension for the Robotlegs-AS3 micro-architecture that makes use of AS3-Signals to trigger commands.

h3. Dependencies

The following external libraries are required by Starling Signals Extension.

* AS3 Signals v.0.9 
* Robotlegs v.1.5.2
* Starling v.1.2
* SwiftSupsenders v.1.6.0
